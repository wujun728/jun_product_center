# Online表单设计工具使用说明

## 概述

Online表单设计工具是一个零代码开发平台，支持快速创建和发布各类表单应用。通过本工具，您可以轻松设计单表、树表、一对一、一对多等多种形式的表单，无需编写代码，无需部署及重启服务即可完成业务功能开发。

## 功能特点

1. **多种表单类型支持**
   - 单表：基础的数据录入表单
   - 树表：支持层级结构的数据管理
   - 一对一：关联两个表的数据编辑
   - 一对多：支持主从表结构的数据管理

2. **丰富的表单组件**
   - 文本框、数字输入、下拉框、日期选择器、时间选择器
   - 复选框、单选框、开关、文本域
   - 文件上传、图片上传、级联选择
   - 支持自定义验证规则和提示信息

3. **灵活的表单设计**
   - 拖拽式表单元素添加
   - 支持复制、编辑、删除表单元素
   - 可自定义表单布局（单列、两列、三列、四列）
   - 支持表单验证模式配置

4. **增强功能**
   - 自定义按钮：添加业务操作按钮
   - JS增强：支持表单事件的JavaScript处理逻辑
   - Java增强：支持服务端业务逻辑扩展

## 使用流程

1. **访问在线表单设计平台**
   打开浏览器，访问系统中的online模块页面：http://your-domain/online/

2. **创建新表单**
   - 点击"新建表单"按钮
   - 填写表单基本信息（名称、编码、类型、布局、描述）
   - 选择表单类型（单表/树表/一对一/一对多）

3. **设计表单**
   - 从左侧组件库选择需要的表单组件，点击添加到设计区域
   - 点击表单元素可进行编辑、复制、删除操作
   - 配置表单验证规则

4. **保存表单**
   - 点击顶部导航栏的"保存"按钮
   - 系统会保存表单配置信息

5. **预览表单**
   - 点击顶部导航栏的"预览"按钮
   - 查看表单在实际使用中的效果

6. **发布表单**
   - 确认表单设计完成后，点击"发布"按钮
   - 表单即可投入使用，无需部署及重启服务

## 访问方式

- 表单设计平台：http://your-domain/online/
- 表单设计器：http://your-domain/online/form.html

## 注意事项

1. 表单编码必须唯一，不能与已有的表单重复
2. 对于关联表结构（一对一、一对多），需要先确保关联的表已存在
3. 发布后的表单可以直接在系统中使用，无需额外部署
4. 支持对已发布的表单进行修改和重新发布

## 示例场景

1. **基础信息录入**：使用单表类型创建员工信息、客户信息等基础数据录入表单
2. **分类管理**：使用树表类型创建组织架构、商品分类等层级数据管理表单
3. **订单管理**：使用一对多类型创建订单及订单明细管理表单
4. **用户档案**：使用一对一类型创建用户基本信息与详细档案管理表单
