##### ESLint代码严格检查

1. 运行报错很多，本来打算关闭ESLint，空格都报错，后来找到ESLint自动格式化代码的方法就好了，可以不用关闭ESLint
2. 在代码前加上这个注释，这块代码规则的校验就会跳过
```JavaScript
/* eslint-disable no-new */
```
