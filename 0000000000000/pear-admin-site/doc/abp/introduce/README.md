## 项目介绍  :id=start

Pear Admin Abp 基于 AspNetBoilerplate 的快速开发平台，使用Layui作为前端呈现，封装常用的功能。旨在设计快速入手，快速实践框架。

> 当前版本：`Release v1.1.0`，查看 [在线演示](http://net.pearadmin.com)。
