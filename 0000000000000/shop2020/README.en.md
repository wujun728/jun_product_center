# java商城+vue后台+uniapp

**项目介绍**

---

基于SpringBoot的后台管理系统，实现了系统权限、动态菜单，用户权限，数据字典等基础功能。基于uniapp的商城移动端，实现了商品分类，用户注册和登录，下单和购物车等操作。
