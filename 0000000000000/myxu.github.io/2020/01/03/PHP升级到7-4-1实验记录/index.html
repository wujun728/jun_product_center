实验环境

  • 腾讯云CentOS7服务器
  • root身份登录

#1、安装所需的EPEL、REMI源

CentOS7自身的源并没有最新的PHP7.4.1包，我们需要而外从其他源中获取，所以首先要安装这些第三方的源。

#2、启用remi源

进入到/etc/yum.repos.d/目录，你会发现remi源提供了多个版本的PHP包，这里我们只要启用7.4版本的就行了。
